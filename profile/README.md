# Sparky Venture Labs

The lab under [Circuvo](https://github.com/circuvo) where things actually get built — iOS apps, macOS tools, and open-source utilities from the world of AV integration and homelab tinkering.

App source stays private. What's public here is meant to be genuinely useful to other developers, integrators, and homelab folks.

### Homelab & AV Automation

| Repo | What it is |
|---|---|
| [homelab-runbook](https://github.com/Sparky-Venture-Labs/homelab-runbook) | Battle-tested commands for a homelab + high-end AV install — UniFi Access, Savant, networking, 3D printing, automation |
| [unifi-configs](https://github.com/Sparky-Venture-Labs/unifi-configs) | Sanitized UniFi VLAN layouts, firewall rules, and access-control scripts |
| [savant-snippets](https://github.com/Sparky-Venture-Labs/savant-snippets) | Savant programming snippets, sclibridge state commands, and device integrations |

### macOS Tools

| Repo | What it is |
|---|---|
| [mac-automations](https://github.com/Sparky-Venture-Labs/mac-automations) | Background automations with launchd + fswatch — auto-unzip downloads, screenshot organizer |
| [dot-files](https://github.com/Sparky-Venture-Labs/dot-files) | Apple Silicon Mac setup — zsh config, Brewfile, macOS defaults |

### iOS Development

| Repo | What it is |
|---|---|
